The Footer
A container for data about data (metadata) placed between the opening HTML tag and the body tag. The body defines the document's body. Meaning, this is the container for the HTML's content The Footer tag defines a footer for a document or Section. Will typically be at the bottom of the page; there can be more than one footer in a document.
Elements you will typically see in the head of the HTML document are:
  • < title > < /title >
  • < meta >
  • < link href="Some Source" >
  • < style > < /style >
The body element contains all the contents of an HTML document, such as:
  • headings < h1 > < /h1 >
  • paragraphs < p > < /p >
  • tables < tables > < /tables >
  • images < img src="Source" >
  • lists < ol > < /ol > < ul > < /ul >
 Some information that is typically included in the footer are:
  • Authorship information
  • Copyright Information
  • Contact Information
  • sitemap
  • Back to top links
  • related documents
